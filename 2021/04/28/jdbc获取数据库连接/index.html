JDBC获取数据库连接

发表于2021-04-28|更新于2021-04-28|java
|字数总计:1.2k|阅读时长:4分钟|阅读量:

获取数据库连接

  1. Driver接口实现类:驱动
    • mysql 8.0驱动 com.mysql.cj.jdbc.Driver
    • 加载与注册JDBC驱动
      • 加载驱动:加载JDBC驱动需调用Class类的静态方法forName(),向其传递要加载的JDBC驱动的类名
        • Class.forName(“com.mysql.cj.jdbc.Driver”) /* 反射 */
      • 注册驱动:DreiverManager类是驱动程序管理器类,负责管理驱动程序
        • 使用DriverManage.registerDriver(com.mysql.cj.jdbc.Driver)注册驱动
        • 通常不显示调用DriverManager类的registerDriver方法来注册驱动程序类的实例?
    • 数据库厂商根据jdk的Driver接口编写Driver的实现类

2 URL

  • JDBC URL用于标识一个被注册的驱动程序,驱动程序管理器通过这个URL选择正确的驱动程序,从而建立到数据库的连接

  • JDBC URl的标准由三部分组成,各部分间用冒号分隔

    • jdbc:子协议:子名称
    • 协议:JDBC URL中的协议总是jdbc
    • 子协议:子协议用于标识一个数据库驱动程序
    • 子名称:一种标识数据库的方法。子名称额可以依不同的子协议而变化,用子名称的目的是为了定位数据库提供足够的信息。包括主机名(对应的服务端的ip地址),端口号,数据库名
      • 举例
      • jdbc:mysql://localhost:3306/test?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone = GMT
      • 在 mysql 8.0应该这样写,加上时区

  • 几种常用的数据库的jdbc url

    • mysql连接URL编写的方法
      • jdbc:mysql://主机名称:mysql服务端口号/数据库名称?参数=值&参数=值
      • jdbc:mysql://localhost:3306/test
      • jdbc:mysql://localhost:3306/test?useUnicode=true&characterEncoding=utf8(如果jdbc程序与服务器段的字符集不一致,会导致乱码,那么可以通过参数指定服务器端的字符集)
      • jdbc:mysql://localhost:3306/test?user=root&password=123456
    • Oracle 9i的连接URL编写方式：
      • jdbc:oracle:thin:@主机名称:oracle服务端口号:数据库名称
      • jdbc:oracle:thin:@localhost:1521:atguigu
    • SQLServer的连接URL编写方式：
      • jdbc:sqlserver://主机名称:sqlserver服务端口号:DatabaseName=数据库名称
      • jdbc:sqlserver://localhost:1433:DatabaseName=atguigu

  1. 用户名和密码

    • user,password可以用”属性名=属性值”方式告诉数据库
    • 可以调用DriverManager类的getConnection()方法建立到数据库的连接

      获取数据库连接的方法

  2. 连接的方式1(出现了第三方API)

     //方式一
    Driver driver = new com.mysql.cj.jdbc.Driver();

    String url = "jdbc:mysql://localhost:3306/test?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone = GMT";
    //将用户名和秘密码封装在Properties中
    Properties info = new Properties();
    info.setProperty("user", "root");
    info.setProperty("password", "123456");
    Connection connection = driver.connect(url,info);

    System.out.println(connection);

  3. 连接的方式2(利用反射实现Dynamic动态性,使程序具有更好的可移植性)

         //方式二
     //1.根据反射获取Driver的mysql接口子类实现类
     Class clazz = Class.forName("com.mysql.cj.jdbc.Driver");
     Driver driver = (Driver) clazz.getDeclaredConstructor().newInstance();
     //2.提供要连接的数据库
     String url ="jdbc:mysql://localhost:3306/test?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone = GMT";
     //3.提供连接需要的用户名和密码
     Properties info = new Properties();
     info.setProperty("user", "root");
     info.setProperty("password", "123456");
     //4.获取连接
     Connection connection = driver.connect(url,info);
     System.out.println(connection);

  4. 连接的方式3(利用DriverManager实现的管理Driver)

     //方式三 利用DriverManager类而不用Driver获取连接,需要注册驱动
 //1.根据反射获取Driver的mysql接口子类实现类
 Class clazz = Class.forName("com.mysql.cj.jdbc.Driver");
 Driver driver = (Driver) clazz.getDeclaredConstructor().newInstance();
 //2.提供另外三个连接的基本信息
 String url = "jdbc:mysql://localhost:3306/test?useUnicode=true&characterEncoding=utf-8&useSSL=false&serverTimezone = GMT";
 String user = "root";
 String password = "123456";
 //注册驱动
 DriverManager.registerDriver(driver);
 //获取连接
 Connection connection = DriverManager.getConnection(url,user,password);
 System.out.println(connection);

  5. 连接方式4(优化获取Driver实现类的对象)

     //方式四 优化方式三 优化获取Driver实现类的对象

 //1.加载mysql的Driver

  6. 连接方式5(最终版,将数据库的驱动与程序解耦写入配置文件)

     //方式五 将数据与程序结构,写入配置文件
 //得到properties文件
 InputStream is = ConnectionTest.class.getClassLoader().getResourceAsStream("mysql.properties");
 Properties initial = new Properties();
 initial.load(is);
 String driver = initial.getProperty("driver");
 String url = initial.getProperty("url");
 String user = initial.getProperty("user");
 String password = initial.getProperty("password");

 //加载驱动
 Class.forName(driver);
 //获得连接
 Connection connection = DriverManager.getConnection(url,user,password);
 System.out.println(connection);

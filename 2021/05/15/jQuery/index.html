JQuery基础

  • 概念: 一个Javscript框架,能够简化javascript代码

  • 本质: js文件,封装了js的原生代码

  • 版本

    1. 1.x:
      • 兼容ie,使用最广泛,官方只做BUG维护,功能不再新增。一般项目用1.x版本就可以
      • 最终版本:1.12.4(2016/5/20)
    2. 2.x:
      • 使用最少
    3. 3.x:
      • 最新版本:新浏览器支持
    4. jquery-xxx.js 与 jquery-xxx.min.js区别
      1. jquery-xxx.js:开发版本,良好的缩进与注释,给程序员看
      2. jquery-xxx.min.js:生产版本,程序中使用,没有缩进,体积小,程序加载更快

        快速入门

  • 步骤

  • 导入Jquery文件,导入min.js文件

  • 使用

      var div1 = $("#div1");
  alert(div1.html());

  • 优势

    • jquery对象在操作时更加方便’
    • jquery对象和js对象不通用

  • Jquery与js对象转换

  • jq – > js:jsq对象[索引] 或者 jq对象.get(索引)(jq对象本质上是Object的数组)

  • js – > jq:$(js对象)

           let $div2 = $("div");
       alert($div2.length)
       $div2.html("bbb")
     
       $div2[0].innerHTML = "ccc";
       $div2.get(1).innerHTML = "ddd";

  • 选择器:筛选具有相似特征的元素(标签）

    1. 基本语法学习

      1. 事件绑定
      2. 入口函数
        1. js中的window.onload方法会被下一个函数覆盖
        2. Jquery中的$(function())可以重复
      3. 样式控制
        • $(function(){$(“#div1”.css(“backgroundColour”,”pink”))})

    2. 分类

      1. 基本选择器

        1. 标签选择器（元素选择器）
          • 语法： $(“html标签名”) 获得所有匹配标签名称的元素
        2. id选择器
          • 语法： $(“#id的属性值”) 获得与指定id属性值匹配的元素
        3. 类选择器
          • 语法： $(“.class的属性值”) 获得与指定的class属性值匹配的元素
        4. 并集选择器：
          • 语法： $(“选择器1,选择器2….”) 获取多个选择器选中的所有元素

      2. 层级选择器

        1. 后代选择器
          • 语法: $(“A B”) 选择A元素内部的所有B元素(包括所有后辈)
        2. 自带选择器
          • 语法: $(“A > B”) 选择A元素内部的自带B元素

      3. 属性选择器

        1. 属性名称选择器

          • 语法： $(“A[属性名]”) 包含指定属性的选择器

        2. 属性选择器

          • 语法： $(“A[属性名=’值’]”) 包含指定属性等于指定值的选择器

        3. 复合属性选择器

          • 语法： $(“A[属性名=’值’][]…”) 包含多个属性条件的选择器

        4. 属性选择器里的正则表达式

          1. 开头

            • $(“A[属性^=’ee’]”);

          2. 结尾

            • $(“A[属性$=’ee’]”);

          3. 包含

            • $(“A[属性*=’ee’]”);
            1. 过滤选择器

          4. 首元素选择器

            • 语法： :first 获得选择的元素中的第一个元素

          5. 尾元素选择器

            • 语法： :last 获得选择的元素中的最后一个元素

          6. 非元素选择器

            • 语法： :not(selector) 不包括指定内容的元素

          7. 偶数选择器

            • 语法： :even 偶数，从 0 开始计数

          8. 奇数选择器

            • 语法： :odd 奇数，从 0 开始计数

          9. 等于索引选择器

            • 语法： :eq(index) 指定索引元素

          10. 大于索引选择器

            • 语法： :gt(index) 大于指定索引元素,不等于

          11. 小于索引选择器

            • 语法： :lt(index) 小于指定索引元素,不等于

          12. 标题选择器

            • 语法： :header 获得标题（h1~h6）元素，固定写法

        5. 表单过滤选择器

          1. 可用元素选择器
            • 语法： :enabled 获得可用元素
          2. 不可用元素选择器
            • 语法： :disabled 获得不可用元素
          3. 选中选择器
            • 语法： :checked 获得单选/复选框选中的元素
          4. 选中选择器
            • 语法： :selected 获得下拉框选中的元素

DOM操作

  1. 内容操作

    1. html(): 获取/设置元素的标签体内容

       <a><font>内容</font></a>  --> <font>内容</font>

    2. text(): 获取/设置元素的题纯文本内容,设置text内容时会换掉其所在的标题标签

       获取/设置元素的标签体纯文本内容   <a><font>内容</font></a> --> 内容

    3. val(): 获取/设置元素的value属性值

  2. 属性操作

    1. 通用属性操作

      1. attr(): 获取/设置元素的属性
      2. removeAttr(): 删除属性
      3. prop(): 获取/设置元素的属性
      4. removeProp(): 删除属性
      • attr和prop区别
        1. 如果操作的是元素的固有属性,建议使用prop
        2. 如果操作的是自定义属性,建议使用attr

    2. 对class属性的操作

      1. addClass():添加class属性值
      2. removerClass():删除class属性值
      3. toggleClass():切换class属性值
        • 如果元素对象有class则删除,如果不存在则添加
      4. css():获取/设置css样式

  3. CRUD操作

    1. 添加
      1. 父级元素的添加
        1. append():父元素将子元素追加到末尾
          • 对象1.append(对象2):将对象2添加到对象1元素内部,并且在末尾
        2. prepend():父元素将子元素追加到开头
          • 对象1.prepend(对象2):将对象2添加到对象1元素内部,并且在开头
        3. appentTO():括号里追加调用方法者在末尾
          • 对象1.appendTo(对象2):将对象1添加到对象2内部,并且在末尾
        4. prepentTo():括号里追加调用方法者在开头
          • 对象1.appendTo(对象2):将对象1添加到对象2内部,并且在开头
      2. 兄弟元素的添加
        1. after():将元素添加到元素后面
          • 对象1.after(对象2):将对象2添加到对象1后,为兄弟关系
        2. before():将元素添加到元素前边
          • 对象1.before(对象2):将对象2添加到对象1后,为兄弟关系
        3. insertAfter():插队
        4. insertBefore():插队
    2. 移除
      1. remove():移除元素
        • 对象.remove():将此对象删除
      2. empty():清空元素的所有后代元素
        • 对象.empty():将此元素的所有子节点删除,但保留该节点和属性

this

  • this是dom对象不是jQuery对象,需要$(this)才能调用this的Jquery对象

Jquery高级

  1. 动画

    1. 三种方式显示和隐藏元素

      1. 默认显示和隐藏方式

        1. show([speed,[easing],[fn]])
          1. 参数：
            1. speed：动画的速度。三个预定义的值(“slow”,”normal”, “fast”)或表示动画时长的毫秒数值(如：1000)
            2. easing：用来指定切换效果，默认是”swing”，可用参数”linear”
              • swing：动画执行时效果是 先慢，中间快，最后又慢
              • linear：动画执行时速度是匀速的
            3. fn：在动画完成时执行的函数，每个元素执行一次。
        2. hide([speed,[easing],[fn]])
        3. toggle([speed],[easing],[fn])

      2. 滑动显示和隐藏方式

        1. slideDown([speed],[easing],[fn])
        2. slideUp([speed,[easing],[fn]])
        3. slideToggle([speed],[easing],[fn])

      3. 淡入淡出显示和隐藏方式

        1. fadeIn([speed],[easing],[fn])
        2. fadeOut([speed],[easing],[fn])
        3. fadeToggle([speed,[easing],[fn]])

  2. 遍历

    1. js的遍历方式
    • for(初始化值;循环结束条件;步长)

    1. jq的遍历方式

      1. jq对象.each(callback)
        1. 语法：
          jquery对象.each(function(index,element){});
           * index:就是元素在集合中的索引
 * element：就是集合中的每一个元素对象
 * this：集合中的每一个元素对象
 * function中的index,element固定位置,但是可以省略
        2. 回调函数返回值：
          • true:如果当前function返回为false，则结束循环(break)。
          • false:如果当前function返回为true，则结束本次循环，继续下次循环(continue)
      2. $.each(object, [callback]):可以传递js数组对象,不仅jq对象
      3. for..of: jquery 3.0 版本之后提供的方式
        for(元素对象 of 容器对象)
      4. tips:如果callback()函数里有返回false,则相当于break,如果是true,则相当于continue

    2. 事件绑定

      1. jQuery标准的绑定方式
        • jq对象.事件方法(回调函数);
        • tips:如果调用事件方法,不传递回调函数,则会出发浏览器默认行为。
          • 对象.submit,提交表单
          • 对象.focus(如果没有回调函数,默认设置焦点)
      2. on绑定事件/off解绑事件
        • jq对象.on(“事件名称”,回调函数);
        • jq对象.off(“事件名称”);
        • tips:如果off不传入任何参数,则会默认解绑所有事件
      3. 事件切换:toggle
        • jq对象.toggle(fn1,fn2);
          • 点击会依次循环切换执行函数。
          • jQuery1.9取消此功能,导入jQuery-migrate.js可以执行

    3. 插件:增强JQuery的功能

      1. 实现方式:
        1. $.fn.extend(object)
          • 增强通过Jquery获取的对象的功能 $(“#id”)
        2. $.extend(object)
          • 增强Jquery对象自身的功能 $/jQuery
          • 调用方式$.插件名

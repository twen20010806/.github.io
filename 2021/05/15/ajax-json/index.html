AJAX

1.概念 ASynchronous JavaScript And XML 异步的JavaScript 和 XML
1. 异步,同步:客户端和服务器通信的基础上
1. 同步:客户端必须等待服务器的响应,在等待的期间客户端不能做其他操作
2. 异步:客户端不需要等待服务器端的响应,在服务器处理请求的过程中,客户端可以进行其他操作
* 客户端的各个元素之间异步,互不影响

3. Ajax即Asynchronous Javascript And XML（异步JavaScript和XML）在 2005年被Jesse James Garrett提出的新术语，用来描述一种使用现有技术集合的‘新’方法，包括: HTML 或 XHTML, CSS, JavaScript, DOM, XML, XSLT, 以及最重要的XMLHttpRequest。 使用Ajax技术网页应用能够快速地将增量更新呈现在用户界面上，而不需要重载（刷新）整个页面，这使得程序能够更快地回应用户的操作。

  1. 实现方式

    1. 原生的Js实现方式(了解)

    2. JQuery实现方式

      1. $.ajax()

         $.ajax({
   url:"/Ajax/ajaxServlet",
   type:"post",
   data:{"username":"twen","age":"23"},
   success:function (data){
     alert(data);
   }
 });
        • success:function,error:function里的data为response里的响应内容

      2. $.get()

        • 用来发送get请求
        • 语法:$.get(url,[data],[SuccessCallback],[type])
        • url:请求路径
        • 请求参数
        • 回调函数
        • 响应数据类型
        • 使用get不再需要写键名

      3. $.post()

        • 用来发送post请求
        • 语法:$.post(url(必写),[data],[callback],[type]
      • 常用的键:
        • url
        • type:”post”,”get”
        • data:发送的数据
        • success:function(data)
        • error:function(data)
        • dataType:获取的响应的格式(MIME类型)

JSON

  1. 概念:JavaScript Object Notation JavaScript对象表示法

     var p = ("name":张三,"age":23,"gender":"男);
    • JSON现多用于存储和交换文本信息的语法
    • 进行数据的传输
    • JSON比XML更小,更快,更易解析

  2. 语法:

    1. 基本规则

      1. 数据在名称键值对中:Json数据由键值对构成
        • 键用引号(单双都行)引起来,也可以不使用引号
        • 值的取值类型:
          • 数字(正数或浮点型)
          • 字符型(在双引号中)
          • 逻辑值(true或false)
          • 数值(在方括号中):{“person”:[{},{}]}
          • 对象(在花括号中):{“address”:{“province”:”陕西”,….}}
          • null
      2. 数据由逗号分隔
      3. 花括号保存对象
      4. 方括号保存数组

    2. 获取数据

      1. json对象.键名

      2. json对象[“键名”]

      3. 数组对象[索引]

      4. 对于Json对象和Json数组的遍历

        1. 对象
          ** for中的key是字符串,所以不能用person.key获取数据

           for(key in person){
      alert(person[key]);
  }

        2. 键值对,数组对象的嵌套

           var persons ={
 "person":  [{"name": "张三", age: 23, "gender": true},
       {"name": "李四", age: 24, "gender": false},
       {"name": "王五", age: 25, "gender": true}]
   };
     
 for (let i = 0; i < persons.person.length; i++) {
   var person = persons.person[i];
   for(key in person){
       alert(person[key]);
   }
   }

  3. JSON数据和java对象的相互转换

    1. Json解析器

      • 常见的解析器:Jsonlib,Gson(Google提供),fastjson(阿里),jackson(Spring MVC框架内置)

    2. Json转为Java对象

      1. redValue(json数据,Class)

    3. Java对象转换Json

      1. 使用步骤

        1. 导入jackson的相关jar包

        2. 创建Jackson核心对象ObjectMapper

        3. 调用ObjectMapper的相关方法

          1. 转换方法

            • writeValue(参数1,参数2)
              • 参数1
                • file:将obj对象转换为JSON字符串,并保存到指定的文件中
                • Writer:将obj对象转换为Json字符串,并将json数据填充到字符输出流中
                • OutputStream:子节输出流
            • writeValueAsString(),传入对象将对象转换为Json字符串

          2. 注解:

            1. @JsonIgnore:排除属性
            2. @JsonFormat:属性值的格式化(格式化与DateFormat要求一样)

          3. 复杂的java对象转换

            1. list:数组
            2. Map:和对象格式一致
